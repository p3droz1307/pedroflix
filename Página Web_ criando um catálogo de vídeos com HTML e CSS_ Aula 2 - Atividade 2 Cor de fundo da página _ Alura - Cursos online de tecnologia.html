Transcrição

Tínhamos uma página que funciona e que exibe um título, um parágrafo e um cabeçalho, localizado no topo da página. Além disso, inserimos um vídeo no meio da página.

Mas agora, queremos alterar a aparência dessa página. Não queremos mais o texto com letra preta, queremos mudar a aparência da página. Para isso, vamos trabalhar com o estilo da página.

Não apenas os elementos que aparecem na página, mas vamos alterar o estilo.

Como faremos isso? Clique com o botão direito na sua página, fora do vídeo, e escolha a opção "Inspecionar" (Inspect).

Com isso, será exibido na lateral direita o código HTML da nossa página.

Vai aparecer o bloco body, que contém o cabeçalho, o título, o parágrafo, etc.

Quando você passa o mouse sobre o body, ele destaca a página inteira. Se você passar o mouse sobre o header, ele destaca apenas o cabeçalho.

Queremos alterar o estilo do body inteiro, então vamos selecioná-lo. Na parte inferior, onde está escrito Estilos, podemos ver várias propriedades que podem ser alteradas.

aba "Estilos" da funcionalidade "Inspecionar"

element.style {

}
body {
display: block;
margin: 8px;

}

Queremos adicionar uma cor de fundo preta. Para isso, vamos editar o bloco element.style e digitar background: black;. Agora, a página inteira tem um fundo preto.

element.style {
  background: black;
}
body {
display: block;
margin: 8px;

}

Além do fundo preto, podemos adicionar mais estilos.

Por exemplo, se você notar que o texto está preto sobre um fundo preto, o que torna a leitura difícil, podemos mudar a cor da fonte para branco.

Vamos pressionar a tecla "Tab" e adicionar mais um estilo. Para isso, digite color: white;. Agora, temos a fonte branca.

element.style {
  background: black;
    color: white;
}

No entanto, se sairmos e voltarmos mais tarde, todas as alterações feitas através do "Inspecionar" serão perdidas. Isso ocorre porque essas alterações são feitas apenas no navegador e não são salvas no nosso projeto.

Como podemos aplicar estilos específicos ao nosso body diretamente no nosso arquivo HTML?

Arquivo para estilos (styles)

A boa prática é criar um novo arquivo para os estilos. Vamos criar um novo arquivo chamado styles.css. Nesse arquivo, vamos especificar que o body terá as características de cor e cor de fundo.

Para isso, escrevemos:

styles.css

body {
    color: white;
    background: black;
}

No arquivo CSS, devemos indicar a seção que queremos configurar e inserir as configurações entre chaves.

Para que as alterações sejam aplicadas no HTML, precisamos informar ao nosso arquivo HTML para usar os estilos do arquivo styles.css.

Para fazer isso, precisamos adicionar uma tag link no cabeçalho da página (head) do nosso arquivo HTML. Faremos o seguinte na primeira linha do arquivo index.html:

index.html

<head>
    <link rel="stylesheet" href="styles.css">
</head>

Nesse código, o rel indica a relação do tipo stylesheet (folha de estilos). E href indica a referência para o arquivo em que o estilo está configurado.

Agora, quando atualizamos a página, os estilos do arquivo styles.css são aplicados.

Vamos refinar ainda mais a nossa página.

Primeiro, queremos adicionar um título adequado à aba do navegador. Para isso, adicionamos uma tag <title> no cabeçalho do nosso arquivo HTML:

<head>
    <link rel="stylesheet" href="styles.css">
    <title>Aluraflix</title>
</head>

Agora, o título da aba do navegador será "Aluraflix".

Perceba que, quando fazemos uma única alteração é fácil saber onde está o erro ao testarmos a página no navegador. Por isso é importante testar conforme fazemos as modificações no código. Faça passo a passo conforme vai testando no navegador. Essa é uma boa prática na criação de sites.

Agora, queremos agrupar o título <h1> e o parágrafo <p>, que fazem parte da mesma seção. Para isso, usamos a tag <div>:

<div>
<h1>ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?</h1>
<p>#homem-aranha</p>
</div>

Da mesma forma, vamos inserir o vídeo dentro de uma tag div.

Para melhorar a legibilidade do nosso código, podemos formatar o documento no Visual Studio Code usando o atalho "Ctrl + Shift + P" e digitando "format". Podemos clicar em "Format Document". Dessa forma, ele vai alinhar o text conforme a indentação correta para o código. Assim conseguimos ver mais claramente onde as tags e os elementos estão abrindo e fechando.

Na nossa página teremos vários vídeos, então serão várias seções (uma para cada vídeo). Por isso, vamos criar uma tag chamada section antes das duas divs com com título e após a div do vídeo.

Podemos salvar com o atalho "Ctrl + S", ir para o navegador e atualizá-lo. Está tudo funcionando conforme esperado.

Por fim, queremos informar ao navegador que a nossa página está em português do Brasil. Para isso, adicionamos o atributo lang à tag html. O "lang" é abreviação de language (linguagem):

<html lang="pt-br">
</html>

Agora os navegadores vão entender que essa página está em portugês do Brasil.

A nossa página está pronta e estilizada.
